# Crypto Contribution Targets

This file tracks the contribution focus for crypto, Web3, Stellar, DeFi, and blockchain developer tooling.

## Preferred areas

- Stellar and Soroban developer tooling
- DeFi dashboards, vaults, analytics, and SDK docs
- Ethereum developer documentation
- Web3 client libraries and examples
- Security, testing, and contributor experience for smart contract projects

## Selection criteria

- The repository has recent maintainer activity.
- The issue is unassigned or explicitly open for help.
- The scope is small enough for a focused PR.
- The expected verification is clear.
- The change improves documentation, tests, examples, or developer workflow.

## Avoid for now

- Issues already assigned to another contributor.
- Bounty or wave issues where another applicant has been accepted.
- Large content pages that need maintainer direction before writing.
- Smart contract behavior changes without enough local test context.
