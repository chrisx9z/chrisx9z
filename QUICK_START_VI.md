# Bat dau tren GitHub

## 1. Tao profile repository

Tao repository public voi ten:

```text
chrisx9z
```

Repository profile phai trung voi username thi GitHub moi hien README tren trang ca nhan.

## 2. Commit dau tien

Them file:

```text
README.md
```

Noi dung lay tu file `github-profile/README.md`.

Commit message:

```text
Add profile README
```

Comment ngan cho commit:

```text
Creates a concise profile README so visitors can quickly understand my focus, contribution style, and current goals.
```

## 3. Commit thu hai

Them file:

```text
CONTRIBUTION_PLAN.md
```

Commit message:

```text
Add contribution workflow
```

Comment ngan cho commit:

```text
Defines a repeatable contribution workflow focused on small, reviewable, and verifiable open-source changes.
```

## 4. Commit thu ba

Them file:

```text
COMMIT_COMMENTS.md
```

Commit message:

```text
Add commit comment examples
```

Comment ngan cho commit:

```text
Adds reusable commit comments so each contribution explains its purpose clearly.
```

## Nguyen tac commit

- Moi commit chi nen lam mot viec ro rang.
- Moi commit co mot comment ngan noi vi sao thay doi nay co ich.
- Tranh commit message chung chung nhu `update`, `fix`, `changes`.
